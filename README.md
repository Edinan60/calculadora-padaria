# 🍞 Calculadora de Custos para Padaria

Este projeto é uma **Atividade de Extensão Universitária** desenvolvida para auxiliar microempreendedores do setor de panificação na precificação correta de seus produtos.

## 🎯 Objetivo do Projeto
Muitas padarias de bairro enfrentam dificuldades em calcular o custo real de produção, o que pode gerar prejuízos. Este software permite:
- Calcular o custo proporcional de cada ingrediente.
- Definir margem de lucro desejada.
- Obter o preço de venda sugerido de forma instantânea.

## 🛠️ Tecnologias Utilizadas
- **Python**: Lógica de programação.
- **Streamlit**: Interface web moderna e responsiva.
- **GitHub**: Controle de versão e hospedagem do código.

## 🚀 Como acessar a ferramenta
O projeto está disponível online e pode ser acessado por qualquer dispositivo (computador ou celular) através do link abaixo:
👉 **(https://calculadora-padaria-hcnygvyesklrxhb36u5pzj.streamlit.app/)**

## 📝 Como usar
1. Insira o **nome do ingrediente**.
2. Informe o **preço pago na embalagem** fechada.
3. Informe o **peso/tamanho total da embalagem** (ex: 1000g para 1kg).
4. Informe a **quantidade exata usada na receita**.
5. Clique em **"Adicionar à Receita"**.
6. Repita para todos os itens e ajuste a barra de **Margem de Lucro** para ver o preço final.
